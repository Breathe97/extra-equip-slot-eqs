# extra-equip-slot-eqs 额外的装备栏

> 这是一个为饥荒联机版打造的轻量扩展模组，让角色拥有额外的服装、护符和背包装备槽。

📦 Steam 创意工坊：[额外的装备栏](https://steamcommunity.com/sharedfiles/filedetails/?id=2946486855)

## ✨ 功能简介

- 🧥 增加额外的“服装栏”，用于装备服装类物品
- 📿 增加额外的“护符栏”，用于佩戴护符类物品
- 🎒 增加额外的“背包栏”，用于装备背包类物品
- 🤖 支持智能识别：未手动适配的物品会根据标签或属性自动分配到合适的槽位
- ⚙️ 支持配置开关：可单独开启或关闭各个额外装备栏，以及自动识别功能
- 🛠️ 修复了部分兼容性问题，例如护符、背包贴图和配方显示相关表现

## 📋 默认装备组合

```text
武器 + 护甲 + 头盔 + 服装 + 护符 + 背包
```

> 说明：大理石甲会占据护甲栏，而不是背包栏。

## 🧭 使用方法

1. 安装模组后进入游戏，额外装备栏会自动生效。
2. 你可以在模组设置中控制各个额外槽位是否启用。
3. 对于未适配的物品，模组会优先尝试自动识别并分配到合适的位置。

## 🔧 适配情况

模组已适配部分常见模组物品；如果你希望补充适配，可在创意工坊留言，提供“物品英文名 + 中文名 + 所属模组名称”。
