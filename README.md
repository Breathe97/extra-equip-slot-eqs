# extra-equip-slot-eqs 额外的装备栏

> 这是一个为爱发电的项目。

为人物添加额外的**服装栏**、**护符栏**、**背包栏**，让你的饥荒联机版体验更加丰富。

📦 **Steam 创意工坊**：[额外的装备栏](https://steamcommunity.com/sharedfiles/filedetails/?id=2946486855)

---

## ✨ 功能

- 🧥 **服装栏** — 额外装备槽，用于穿戴服装类物品
- 📿 **护符栏** — 额外装备槽，用于佩戴护符类物品
- 🎒 **背包栏** — 额外装备槽，用于装备背包（可与护甲共存）
- 🤖 **智能适配** — 未手动适配的模组物品会自动分配到合适的装备槽
- ⚙️ **可配置** — 每个装备槽均可独立开关，智能适配也可分别控制

## 📋 默认装备组合

```
武器 + 护甲 + 头盔 + 服装 + 护符 + 背包
```

> 大理石甲会占据护甲栏而不是背包栏。

## 🔧 已适配的模组物品

已适配部分常见模组的物品到对应的装备槽。如果你有想适配的模组，可以在创意工坊留言（物品英文名 + 中文名 + 模组名称）。
