# extra-equip-slot-eqs 额外的装备栏

> 这是一个为饥荒联机版打造的轻量扩展模组，让角色拥有额外的服装、护符、背包和腰包装备槽。

📦 Steam 创意工坊：[额外的装备栏](https://steamcommunity.com/sharedfiles/filedetails/?id=2946486855)

## ✨ 功能简介

- 🧢 **头饰栏** — 用于装备非护甲的头饰类物品（花环、牛帽、眼球伞等）
- 🧥 **服装栏** — 用于装备服装类物品（雨衣、背心、花衬衫等）
- 🔗 **腰包栏** — 用于装备腰包类物品（指南针）
- 📿 **护符栏** — 用于佩戴护符类物品（重生护符、建造护符等）
- 🎒 **背包栏** — 用于装备背包类物品（背包、猪皮包、坎普斯袋等）
- 🤖 **智能识别** — 未手动适配的物品会根据标签或属性自动分配到合适的槽位
- ⚙️ **可配置** — 可单独开启或关闭各个额外装备栏及自动识别功能
- 🛠️ **兼容修复** — 修复了寄居蟹隐士、重生护符、建造护符折扣、指南针 HUD、容器堆叠、复活背包重开等兼容性问题

## 📋 默认装备组合

```text
武器 + 护甲 + 头盔 + 头饰 + 服装 + 腰包（指南针）+ 护符 + 背包
```

## 🧭 使用方法

1. 安装模组后进入游戏，额外装备栏会自动生效。
2. 你可以在模组设置中控制各个额外槽位是否启用。
3. 对于未适配的物品，模组会优先尝试自动识别并分配到合适的位置。

## 🖼 贴图渲染逻辑

### 符号位分工

游戏角色身上有两个相关的符号位，v2.1.0 后交换了二者的用途：

| 符号 | 用途 | 渲染内容 |
|------|------|----------|
| **`swap_body`**（顶层） | 背包贴图 | 所有背包类物品（含光谱背包的变色粒子） |
| **`swap_body_tall`**（底层） | 身体贴图 | 护甲、服装、护符 |

层级关系：`swap_body > swap_body_tall`（背包在上层，身体贴图在下层，互不干扰）

### 为什么这样分配

- 光谱背包（`backpack_labrat`）的变色+发光效果由 `colouradder` 和 `bloomer` 组件实现，这些组件**只作用于 `swap_body`** 符号
- 护符类物品使用 `torso_amulets` 构建系统，通过 `OverrideSymbol` 渲染到指定符号，与原版 `swap_body` 解耦
- `OverrideSymbol`（贴图替换）与 `SetAddColour`（颜色叠加）是独立渲染管线，背包粒子和护甲纹理可共存

### 执行顺序

```lua
-- 头部渲染（头饰栏启用时）
SetHeadSymbol()               → 头部贴图写入 swap_hat
                               → 露顶帽/普通帽分别处理发型显示
                               → 启迪之冠根据理智值跳过/执行渲染

-- 身体/背包渲染（背包栏或头饰栏启用时）
ClearOverrideSymbol("swap_body_tall")   -- 先卸载旧身体贴图
SetBackSymbol()                         → 背包贴图写入 swap_body（含光谱粒子）
SetBodySymbol()                         → 护甲/服装/护符写入 swap_body_tall
SetSymbolExchange(swap_body_tall, swap_body)  → 确认层级
```

> **v2.2.4 变更**：去掉了旧版使用的 `DoTaskInTime(0, ...)` 多帧延迟渲染，改为直接依次调用。因为经过实际验证，贴图覆盖操作不需要跨帧执行也能正确渲染，简化了逻辑并减少了不必要的帧延迟。

### 护符合名渲染

v2.1.0 新增护符合名映射，护符类物品在渲染时自动叠加 `torso_amulets` 构建中的符号（如 `redamulet`、`blueamulet` 等），解决原版护符在非 `swap_body` 下无法显示贴图的问题。

### 独奏乐器特殊处理

v2.1.1 将独奏乐器（onemanband）从腹部栏移至背部栏。onemanband 使用 `swap_body_tall` 作为源符号名，而非其他背部物品的常规 `swap_body`，需要在 `OverrideSymbol` 时特殊指定。

## 🛠 兼容性修复

| 修复项 | 说明 |
|--------|------|
| 🦀 **寄居蟹隐士** | 适配服装栏，使其能正常接收/拒绝外套类物品 |
| 💀 **重生护符** | 从 NECK 槽正确移除并销毁护符，清理残留贴图 |
| 🔨 **建造护符合成折扣** | 劫持 `GetEquippedItem` 回退查询 NECK → BODY |
| 🧭 **指南针 HUD** | 监听腰包栏的装备/卸下事件，正确开关指南针 HUD |
| 📦 **容器堆叠** | 重写 `GetOverflowContainer`，识别所有新增槽位的打开容器 |
| 👻 **鬼魂复活** | 大门/传送门复活后自动重新打开背包容器 |
| 🖥️ **服务端堆叠** | 服务端 `inventory:GetOverflowContainer()` 同样覆盖新增槽位 |

## ⚙️ 配置选项

| 选项 | 默认值 | 说明 |
|------|--------|------|
| 头饰栏 | 关闭 | 是否扩展头饰栏 |
| 服装栏 | 关闭 | 是否扩展服装栏 |
| 腰包栏 | 关闭 | 是否扩展腰包栏 |
| 护符栏 | 开启 | 是否扩展护符栏 |
| 背包栏 | 开启 | 是否扩展背包栏 |
| 自动识别头饰 | 关闭 | 头部物品且无护甲组件的自动归入头饰栏 |
| 自动识别服装 | 关闭 | 有防水/保暖属性的物品自动归入服装栏 |
| 自动识别护符 | 开启 | 有 `amulet` 标签的物品自动归入护符栏 |
| 自动识别背包 | 开启 | 有 `backpack`/`candybag` 标签的物品自动归入背包栏 |
